## Numbers to avoid unless confirmed

These are tempting to say, but should not be used unless verified with actual proof.

- “India’s No. 1 product sampling agency”
- “Largest sampling network in India”
- “Millions of samples distributed”
- “500+ cities” unless confirmed as an actual operating coverage number
- “Guaranteed sales”
- “Guaranteed ROI”
- “X% conversion uplift” without campaign data
- “Best-in-class”
- “World-leading”
- “Top-rated” unless backed by credible proof
- Any client name, case study number, or campaign result without permission

## Rules for using these numbers

- State numbers as facts, not as bragging lines.
- Always keep the context attached to the number.
- Never turn a target into an achievement.
- Never turn a client-specific estimate into a universal rate card.
- If the number is a range, preserve the range.
- If the number is approximate, say “approx.” or “around.”
- If the number is internal, do not use it publicly unless I ask.
- If the number is stale, ask for the latest update instead of using it.
- If the AI cannot find the number in this file, it should say: “I don’t have a confirmed number for that yet.”
- Review this file quarterly.
